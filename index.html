Presenter par Diagui TOUNKARA

Contexte

De nos jours, le monde connaît des avancées technologiques considérables dans tous les secteurs et cela grâce à l'informatique, qui joue un rôle important dans le développement de nombreuses entreprises et organisations. La nouvelle logique d'organisation du travail impose aux établissements d'enseignement et d'apprentissage d'utiliser essentiellement l'information comme matière première afin d'être plus efficaces. Elles doivent donc intégrer le développement du système d'information dans leurs investissements stratégiques, dans la mesure où elles structurent la saisie, le stockage, l'organisation et la communication de l'information.

Cependant, j'ai constaté qu'une plus grande partie de traitement des données dans les écoles maliennes (primaire, seconaire et lycée) se fait manuellement ce qui génère un certain nombre de problèmes tels que :

  • Des erreurs dans l'établissement des documents lorsqu'il y a un grand nombre d'informations à traiter manuellement
  • Une perte de temps lorsqu'il faut chercher dans le registre
  • L'insécurité de l'information
  • Possibilité d'erreur dans le remplissage des différents documents et registres (exemple: liste de présence
  • Possibilité d'erreur dans le calcul des statistiques ou des notes des élèves
    • Pour resoudre ces problèmes, j'ai décidé de créer une application web de gestion pour les écoles.

    Problèmatique

    Comment faire pour permettre aux établissements scolaires maliennes de gérer non seulement les élèves les enseignants mais aussi les parents d'élèves ?

    Solution

    Mon projet doit gérer tout ce qui concerne les élèves, les parents d’élèves, les absences, les enseignants, le personnel administratif, le matériel, les salles et les horaires. L'objectif est de gagner du temps en automatisant les tâches, en centralisant les données, en facilitant la recherche, l’apprentissage et en sécurisant l'accès aux données. La gestion des élèves est l'admission de ces derniers dans une classe supérieure ainsi que la gestion de leurs déplacements au sein de l'école. A leur entrée, les élèves sont accueillis dans l'école, en fonction de leurs admissions. L'élève est un élément essentiel pour l'existence et le fonctionnement d'une école ou d'un centre de formation, ou il doit avoir un dossier qui lui est propre.

    Besoins fonctionnels

    Les besoins fonctionnels englobent les fonctionnalités de base que devra offrir ma solution, ainsi elle devra permettre à chaque utilisateur de s’authentifier car chaque utilisateur aura un login et un mot de passe unique pour accéder à cette application.

  • L’administrateur peut ajouter, supprimer, modifier ses données, gérer les élèves, les enseignants et les parents d’élèves;
  • Les élèves pourront faire des consultations sur les cours, le calendrier, les résultats trimestriels, les absences et de pouvoir téléchargé les bulletins de notes mais ne peuvent rien modifier, ajouter ou supprimer ;
  • Les enseignants pourront consulter sur les élèves, les modules, les notes des élèves et les absences ;
  • Les parents d’élèves leur rôle est le suivi total sur leurs enfants (notamment la consultation des absences, des bulletins et le calendrier) ;

    • Besoins techniques

    L’interface de mon application doit répondre à un minimum de règle de conception des interfaces homme machines, en plus pour un usage personnel et une meilleure confidentialité, des notions de sécurités s’imposent :

  • L’application doit permettre de gérer les accès des utilisateurs selon un privilège et unétat d’activation de chaque compte ;
  • Il garantit la sécurité d’accès à l’espace administrateur afin de protéger les données personnelles des utilisateurs ;
  • Prévoit contre l’accès direct avec les liens URL et définir un délai de temps pour la fermeture de session non active ;
  • L’interface de cette application doit être ergonome, conviviale voire apte à aider l’utilisateur à mieux gérer son espace de travail (notamment la documentation sur l’application).

    • Lors de la réalisation de cette application, je souhaiterais utiliser les outils et les technologies tels que : Visual Studio Code, le langage HTML, CSS, JavaScript, le Framework Bootstrap, JQuery, MediaQuery, Laravel, le langage PHP/MySQL et enfin le navigateur Google chrome.
